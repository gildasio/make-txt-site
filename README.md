# Make TXT Site

A 'static site generator' made using `make` command to generate a site as txt
files.

## Features

- Atom feed of posts
- GPG sign files
- Hidden posts
- Not a bunch of dependencies, only common and simple GNU/nix environment tools
- CLI browsers friendly
- Responsive
- Simple to host
