---[[ 0x02 Dependencies ]]------------------------------------------------------

*Make TXT Site* only use simple and common tools in a GNU/*nix environment.

Here is a comprehensive list of tools used:

- bash
- cat
- cut
- date
- find
- gpg
- grep
- head
- make
- make
- sed
- sort
- tail
- tree

---[[ 0x03 Bootstrap ]]---------------------------------------------------------

You need to get the project locally and change some variables content to get
start.

------[[ 0x03.00 Download ]]----------------------------------------------------

You can use git:

```
git clone https://github.com/gildasio/make-txt-site
```

Or download a zip file:

```
wget https://github.com/gildasio/make-txt-site/archive/master.zip
```

------[[ 0x03.01 Configure Variables ]]-----------------------------------------

Edit `Makefile` change init variables values.

Variables for general purpose:

- `PREFIX`: Directory where txt content files are in. `public` for example to
  use in Gitlab Pages more easily
- `PUBLIC_INDEX`: Name file (without extension) of public index page. It will be
  used to generate `PUBLIC_INDEX.txt` and `PUBLIC_INDEX.html` files
- `HIDDEN_INDEX`: Name file (without extension) of hidden index page. It will be
  used to generate `HIDDEN_INDEX.txt` and `HIDDEN_INDEX.html` files
- `HIDDEN_PATTERN`: Some pattern to *Make TXT Site* identify if a file is a
  public post or a hidden one

Variables to use only in Atom feed:

- `URL`: URL of your site. Remember to escape / using \
- `AUTHOR`: Your name my fellow
- `SITE`: Site name

---[[ 0x04 Templates ]]---------------------------------------------------------

After configure init variables you have to configure some templates that is in
`template` directory.

------[[ 0x04.00 Files ]]-------------------------------------------------------

- `post.txt`: Template used by `make post <filename>` as a post skel

- `index.txt` and `index.html`: Have to match with `PUBLIC_INDEX` variable in
  `Makefile`. If you change there you have to change here too.

- `priv.txt` and `priv.html`: Have to match with `HIDDEN_INDEX` variable in
  `Makefile`. If you change there you have to change here too.

------[[ 0x04.01 Variables ]]---------------------------------------------------

In `post.txt` template you can put `{{date}}` anywhere in content and when run
`make post <filename>` it will be replaced by today date.

In indexes templates you have to put `{{content}}` somewhere to be replaced by a
file lists, as a `tree` output.

---[[ 0x05 Posts ]]-------------------------------------------------------------

After configure your project (variables and templates) you are ok to go and
create posts.

You can run `make post <filename>` and a filename named <filename> will be
created in `PREFIX` (defualt: `public`) directory as an instance of `post.txt`
template and will open it in your `$EDITOR`.

Otherwise you can simple create a txt file inside `PREFIX` directory and edit
it as well.

After that, run `make all` to create all files.
