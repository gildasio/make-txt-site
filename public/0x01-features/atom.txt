---[[ 0x02 Atom Feature ]]------------------------------------------------------

There is a default template in `template` directory which is a template to form
`entry` element in an Atom Feed scheme.

Running `make feed` *Make TXT Site* will create two files:

- atom.xml
- priv_atom.xml

The difference between then is that priv_atom.xml includes hidden posts as well.

By default *Make TXT Site* create the atom feed file with the last 5 files
updated. If you want to change that you have to modify the number in `tail`
command at `FEED_POSTS` and `HIDDEN_FEED_POSTS` variables pipelines in Makefile.

---[[ 0x03 Validation ]]--------------------------------------------------------

*Make TXT Site* feature follow RFC 4287, Atom format specification. You can
validate it here:

- https://validator.w3.org/feed/
