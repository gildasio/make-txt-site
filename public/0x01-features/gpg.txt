This topic explain how *Make TXT Site* use GPG improving user security.

- 0x04 Signature

---[[ 0x02 Signing ]]-----------------------------------------------------------

As *Make TXT Site* works with txt files it's to sign a post file.

To use this feature you need to have a key configured in your keyring GPG file.

After create and edit your post you can run `make sign <filename>`.

This command will generate a sign to that file and append it to the same file.
This way it's a standalone file and its signature.

> You can sign index.txt files as well.

---[[ 0x03 Verify ]]------------------------------------------------------------

After signing a post/file your users can verify that file to improve integrity.

To do so, there is a simple command:

```
curl $POST_URL | gpg --verify
```
